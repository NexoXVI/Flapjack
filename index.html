Objetivo del sitio web

El objetivo de la página es guardar todo lo que vaya aprendiendo de cada framework, lenguaje de programación, etc. Dentro de esta página estará la sintaxis básica de cada componente de los frameworks de HTML, CSS y JavaScript, incluyendo también su funcionamiento y modificaciones posibles, todo esto a modo de bitácora y fuente para facilitar mi rapidez al programar.

Haré la actualización de cada página individual al final de cada dia, agregando secciones en una misma página o agregando páginas individuales si asi lo requiero. Voy a corregir errores en mi documentación, complementar la información puesta en ella y agregare contenido faltante.

¿Para quien estará disponible?

Por ahora, el sitio web estará disponible para mi y para quienes posean acceso al enlace, esto con el fin de garantizar la máxima funcionalidad del sitio web y además, que el sitio web este disponible para correcciones, sugerencias, etc
